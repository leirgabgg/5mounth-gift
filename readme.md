## 📖 Sobre o Projeto

O **Timeless** é uma aplicação web interativa e romântica criada especialmente para celebrar **5 meses de namoro**. Cada elemento foi pensado com carinho para eternizar os momentos especiais e criar uma experiência única e memorável.

### ✨ Características Especiais

| Característica | Descrição |
|----------------|-----------|
| ⏰ **Contador de Tempo** | Mostra exatamente quanto tempo passou desde o primeiro encontro (anos, meses, dias, horas, minutos, segundos) |
| 📅 **Linha do Tempo** | Uma jornada visual pelos 5 meses, com momentos marcantes de cada etapa |
| 📸 **Galeria de Memórias** | Fotos especiais com efeitos 3D e modal interativo |
| 📦 **Cápsula do Tempo** | Mensagens especiais que só podem ser abertas em datas futuras |
| 🎯 **Quiz do Amor** | Teste de conhecimento sobre a história do casal |
| 🎨 **Tema Claro/Escuro** | Interface adaptável ao gosto do usuário |
| 📱 **100% Responsivo** | Funciona perfeitamente em celular, tablet e desktop |
| ✨ **Animações Fluidas** | Efeitos visuais e partículas que encantam |
